# 字节跳动发布豆包编程模型 Doubao-Seed-Code

> **原文**: [IT之家](https://www.ithome.com/0/896/581.htm)

## 核心内容

- 火山引擎于 2025 年 11 月 11 日正式发布豆包编程模型 Doubao-Seed-Code
- 专为 Agentic Coding（智能体编程）任务深度优化
- 原生支持 256K 长上下文，轻松处理长代码文件和多模块依赖
- 在 SWE-Bench-Verified 基准上达到 SOTA（78.80%）
- 综合成本比行业均价低 62.7%，国内最低价

## 技术解读

### 核心技术突破

**Agentic Coding 架构**：
- "先规划，再开发"模式：助手先提出结构化计划，增量执行，自检，测试失败时自动修复
- 不仅仅是更快的自动补全，而是能够规划、提问澄清、生成测试并迭代修复的 Agent 工作流

**大规模 Agent 强化学习训练系统**：
- 覆盖 10 万+ 容器镜像的海量训练数据集
- 支持 1 万+ 并发沙箱会话
- 单 RL 任务可在数千 GPU 上高效训练
- 通过端到端强化学习实现顶级 Agent 能力，无需蒸馏或标注冷启动数据

**256K 原生长上下文**：
- 轻松处理长代码文件
- 支持多模块依赖的端到端自主编程
- 出色的长上下文理解、任务规划、代码生成和调试能力

### 基准测试表现

| 基准测试 | Doubao-Seed-Code + TRAE | 说明 |
|---------|------------------------|------|
| SWE-Bench Verified | 78.80% | SOTA，展示模型与工具协同的威力 |

### 与竞品对比（成本）

以相同 Token 用量（0-32k 输入）创建交互式英语学习网站为例：

| 模型 | 费用 |
|-----|------|
| Claude Sonnet 4.5 | 约 4.05 元 |
| GLM-4.6 | 约 0.77 元 |
| Doubao-Seed-Code | 约 0.34 元 |

综合成本比行业均价低 62.7%，为国内最低价。

### 工具兼容性

**原生 Anthropic API 支持**：
- 对于使用 Claude Code 的开发者，迁移成本几乎为零
- 只需修改配置文件中的几行代码

**主流开发环境支持**：
- Cursor
- Cline
- Codex CLI
- TRAE（字节自研）
- 其他主流智能编程环境

### 对开发者/行业的影响

1. **成本革命**：以极低成本提供 SOTA 级编程能力，大幅降低 AI 编程门槛
2. **Agent 范式**：推动编程从"补全"向"自主 Agent"转变
3. **国产替代**：为国内开发者提供 Claude Code 的高性价比替代方案
4. **生态整合**：与 TRAE 深度集成，展示模型+工具协同的最佳实践

### 可用性

- 火山方舟平台 API
- 编程套餐订阅：首月 9.9 元起
- 支持所有主流智能编程 IDE 和工具
