# Anthropic 收购 Bun，Claude Code 年化收入突破 10 亿美元

> **原文**: [Anthropic](https://www.anthropic.com/news/anthropic-acquires-bun-as-claude-code-reaches-usd1b-milestone)

## 核心内容

- **里程碑时刻**：2025 年 11 月，Claude Code 公开发布仅 6 个月后，年化收入突破 10 亿美元
- **首次收购**：这是 Anthropic 自 2021 年成立以来的首次收购，交易金额据报道在低数亿美元区间
- **Bun 简介**：
  - 由 Jarred Sumner 于 2021 年创立
  - 全能型 JavaScript 工具包：运行时、包管理器、打包器、测试运行器
  - 性能显著优于竞品（Node.js）
  - 收购时月下载量达 720 万，环比增长 25%
- **战略意图**：Bun 将成为 Claude Code、Claude Agent SDK 及未来 AI 编程产品的底层基础设施
- **开源承诺**：Bun 保持开源（MIT 许可证），原团队继续开发
- **标杆客户**：Netflix、Spotify、KPMG、欧莱雅、Salesforce
- **Anthropic 收入增长**：2025 年 1 月年化收入 10 亿美元 -> 8 月 50 亿美元 -> 10 月 70 亿美元
- **IPO 计划**：据报道 Anthropic 正在进行 IPO 讨论，目标估值 3000 亿美元

## 解读

这笔收购传递了一个清晰信号：Anthropic 正在押注"AI 编程"将成为公司未来最重要的增长引擎。

**Claude Code 的爆发式增长**：6 个月从 0 到 10 亿美元 ARR，这个增速在 SaaS 历史上几乎没有先例。这说明 AI 编程助手已经从"尝鲜工具"变成了"生产力必需品"。Netflix、Spotify 等顶级科技公司的采用，验证了 Claude Code 在企业级场景的价值。

**为什么是 Bun？**：Bun 的核心优势是速度——它比 Node.js 快得多。对于 AI 编程工具来说，底层运行时的性能直接影响用户体验。更快的代码执行、更快的依赖安装，意味着 AI 生成的代码能够更快地被验证和迭代。此外，Bun 的一体化设计（运行时+包管理+打包+测试）简化了工具链复杂度，非常适合 AI Agent 自动化操作。

**首次收购的战略信号**：Anthropic 成立 4 年来首次出手收购，说明公司已经从"纯研发模式"转向"平台构建模式"。当你开始通过收购来补强技术栈，意味着你在为长期竞争做准备。

**开源策略的延续**：保持 Bun 开源是明智之举。开发者社区的信任是 Bun 最大的资产，闭源化将毁掉这一切。Anthropic 需要的是 Bun 的技术和团队，而非对一个开源项目的独占控制。

从更宏观的视角看，Claude Code 的成功正在重塑 Anthropic 的商业模式。当 AI 编程工具能够单独贡献 10 亿美元收入时，它就不再只是 Claude API 的附属产品，而是一个独立的增长极。
