# DeepSeek 发布 V3.2，持续引领开源大模型

> **原文**: [DeepSeek API Docs](https://api-docs.deepseek.com/news/news251201)

## 核心内容

- DeepSeek 于 2025 年 12 月 1 日发布 DeepSeek-V3.2 和 DeepSeek-V3.2-Speciale 两款模型
- V3.2 是 V3.2-Exp 的正式继任者，在推理速度和推理能力之间取得平衡
- V3.2-Speciale 提供最强推理能力，可与 Gemini-3.0-Pro 媲美，目前仅支持 API 访问
- 685B 参数的稀疏注意力模型，支持长上下文和集成 Agent 工具

## 技术解读

### 核心技术突破

**DeepSeek 稀疏注意力 (DSA)**：首次实现细粒度稀疏注意力机制，在长上下文训练和推理效率方面取得显著提升，同时保持几乎相同的模型输出质量。

**大规模 Agent 训练数据合成**：新方法覆盖 1,800+ 环境和 85,000+ 复杂指令。这是 DeepSeek 首个将思考直接集成到工具使用中的模型，支持思考模式和非思考模式下的工具调用。

### 基准测试表现

| 基准测试 | DeepSeek-V3.2 | 对比模型 |
|---------|---------------|---------|
| 2025 AIME | 96.0% | GPT-5 High: 94.6% |
| HMMT 2025 | 99.2% | Gemini 3 Pro: 97.5% |
| IMO/CMO/ICPC/IOI 2025 | 金牌级别 (V3.2-Speciale) | - |

### 与竞品对比

相比 GPT-5，DeepSeek-V3.2 在数学推理任务上表现更优，同时成本仅为十分之一。根据 Introl 的分析，V3.2 达到了"GPT-5 级别性能"，同时保持了极高的成本效率。

### 对开发者/行业的影响

1. **开源生态领先**：继续巩固 DeepSeek 在开源大模型领域的领先地位
2. **Agent 能力增强**：原生支持工具调用和 Agent 工作流，为开发者构建智能代理提供强大基础
3. **成本优势明显**：以更低的成本提供接近顶尖闭源模型的性能，降低 AI 应用门槛
4. **多平台支持**：已入驻 Microsoft Foundry、NVIDIA NIM 等主流平台，便于企业集成部署

### 可用性

- DeepSeek-V3.2：已在 App、Web 和 API 上线
- DeepSeek-V3.2-Speciale：通过临时端点提供服务，与 V3.2 定价相同，不支持工具调用
