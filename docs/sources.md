::: tip 信息筛选原则
1. **官方优先** - 优先采用公司官方发布的信息
2. **交叉验证** - 重大新闻从多个来源交叉验证
3. **注明出处** - 所有资讯标注原始来源链接
4. **时效标注** - 明确标注信息发布时间
:::
