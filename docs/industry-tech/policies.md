## 主要监管框架

### 欧盟 AI Act

- **状态**: 已生效，分阶段实施
- **核心**: 基于风险的分级监管
- **影响**: 全球AI企业合规标杆

### 中国AI治理

- **生成式AI管理办法**: 2023年8月生效
- **算法推荐管理规定**: 持续完善
- **大模型备案制度**: 上线运营前需备案

### 美国监管动态

- 行政令推动AI安全评估
- 出口管制持续收紧
- 各州立法活跃（加州、纽约等）

## 合规要点

::: warning 创业者注意
AI创业企业应重点关注：
1. **数据合规** - 训练数据来源合法性
2. **内容安全** - 生成内容的安全审核
3. **隐私保护** - 用户数据处理合规
4. **透明度** - AI使用的披露要求
:::
