# SIGNAL — AI播客收录站

YouTube 上最有价值的 AI 播客，翻译、深度解读为中文长文。

## 收录频道

Lex Fridman / All-In Podcast / 20VC / No Priors / BG2 Pod / Lenny's Podcast / Lightcone (YC) / Logan Bartlett

## 项目结构

```
signal/
├── product-preview.html   # Landing page
├── app.html               # 文章列表页
├── article.html           # 文章详情页
├── articles-data.js       # 文章数据
├── login.html             # 登录页（暂未启用）
├── skills/                # Claude Code skills
└── scripts/               # 数据更新脚本
```

## 文章格式

每篇解读包含：总结 → 核心洞察(Insight) → 全文意译 + 名词解释
